on distingue dans wordpress 4 grands types de page : 1 page d'accueil : la première page du site par défaut : cette page va lister l'ensemble des articles du site du plus récent au plus ancien ( 5 articles ) 2 page de catégories page de liste qui va regrouper toutes articles qui sont associés à une catégorie (que vous avez sélectionné dans le back office) 3 page d'article pages d'article dans lesquels les internautes vont pouvoir lire le contenu en intégralité ET pouvoir laisser un commentaire dans le front => si vous allez en bas de votre article => formulaire dans lequel les utilisateur peuvent laisser un commentaire un article (post) peut être associé à une catégorie ou à une étiquette 4 page de type page page sont des page dans lesquels les utilisateurs vont pouvoir lire du contenu (et c'est tout) une page ne peut pas être associées à une catégorie ou à une étiquette // extension : lms learndash = fonctionnalité supplémentaire // exemple => FreeMium => extension payer // version gratuite
